# Grafana Dashboard for OTRS
This is my [Grafana](https://grafana.com/) Dashboard for [OTRS](https://community.otrs.com/).

# Requirements
- Grafana v6
- Grafana Plugin "grafana-clock-panel"
  
# Import Dashboard
At Grafana:

1. Create > Import
2. Paste JSON or upload the JSON file
